Travel Booking Portal
TripService: Manages trip planning, bookings, and user profiles.
FlightService: Handles flight schedules, availability, and API for flight info.
CancellationService: Processes cancellations and notifies TripService via message broker.


TripService FR
	CRU Endpoints for "User" entity
	CRU Endpoints for "Trip" entity
	CRU Endpoints for "Booking" entity
	After recieving message from CancellationService mark booking as canceled, and mark trip as required attention, decrease/increase flight availability
	Healthcheck endpoint
TripService NFR
	Service should be live 90% of time (up-time)
	Service should return 2**, 4** responses 9/10 times

FlightService FR
	CRU for "Flight" entity
	Endpont for decrease/increase flight availability(seatsBooked/seatsTotal)
